# Day 10 - DATA


## Key Learnings

  
- Categories of data:
  1.Data about people(personally identifiable information)
  2.Data that proves you are you(Credentials. Passwords, password hashes, session tokens, API keys,        SSH keys,etc.)  
  3. Data that belongs to an organisation(Proprietary data. Source code, internal documents, financial      records)
- The more permanent the data, the more valuable it is to steal
